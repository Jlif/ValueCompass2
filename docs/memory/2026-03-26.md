## 关键决策记录

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 跨平台框架 | Tauri | 安装包小(5-10MB vs 150MB+)，内存占用低，Rust原生 |
| 数据服务 | AKTools | 官方封装，维护成本低，HTTP接口易调试 |
| K线存储 | 实时拉取 | 避免首次同步2-3GB数据，按需加载 |
| Python部署 | 内嵌打包 | 用户无感知，随应用启动关闭 |
| 数据库 | SQLite (rusqlite) | 零配置，单文件，适合桌面应用 |

## 技术要点

- **AKShare 限制**: Python独占库，Rust无法直接调用
- **AKTools 能力**: 将 AKShare 函数暴露为 HTTP API，无额外频率限制
- **Tauri 优势**: 相比Electron体积小10-20倍，内存占用少200-400MB
- **数据量级**: A股约5000只，10年K线约1200万条（2-3GB），不适合全量本地存储
- **SQLite 绑定**: 使用 bundled 特性避免系统依赖
