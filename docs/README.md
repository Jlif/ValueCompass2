# 文档索引

本文档目录包含项目所有设计和规划文档。

## 子目录说明

| 目录 | 用途 |
|------|------|
| `memory/` | 工作日志，按日期记录每日进展 |
| `requirements/` | 需求文档，三位数字编号如 `001-需求名称.md` |
| `features/` | 功能文档，三位数字编号如 `001-功能名称.md` |
| `postmortem/` | 复盘记录，沉淀踩坑教训 |

## 文档规范

- 需求文档使用三位数字编号：`001-需求名称.md`
- 功能文档使用三位数字编号：`001-功能名称.md`
- 各子目录包含自己的 README.md 索引
